[TE]
Transposable element data.
A one-line heading is assumed.

[ROI]
Regions of interest data.
A one-line heading is assumed.

[OUTPUT]
Path to and name of output file.
File suffix is arbitrary.

[CONFIG]
See 'example.conf'for an in-depth
explanation of config file usage
and syntax.

This field is optional.

[ANALTYPE]
1. Determine density of repeats
within genome.

2. Determine density of repeats
overlapping with regions of 
interest within genome.

3. Evaluate density of each repeat
across each region of interest.

[REGIONLENGTH]
The number of base pairs in the region of interest.
Use this to generate a .tbl file.
Ignored otherwise.
